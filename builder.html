How Test Mode works

  1. Click Test in Syncr to open the extension's Test Mode.
  2. Drop the downloaded folder in, or the .zip.
  3. Open your site — the activity fires exactly like a shipped one.

Files exported: metadata.json, scraper.json, presence.js, and your logo if provided. These match the exact format Syncr's real activities use — the same layout as native-host/activities/<id>/.

No Discord app? No problem — leave the Application ID blank and your activity uses Syncr's shared default. Bring your own from discord.com/developers later for a custom name and art assets.
