Setup

Install in four steps

Syncr has two parts: the browser extension and a small native host that talks to Discord. You install both once.

1

Grab the latest release

Download the release bundle from GitHub. It contains the browser extension and the native host installer for Windows and Linux.

2

Install the native host

Run the host installer for your platform. This is the small local app that hands your activity to the Discord desktop app over native messaging. Nothing is sent to any server.

3

Add the extension to your browser

Load the Syncr extension in any Chromium or Mozilla based browser. Make sure the Discord desktop app is running so Syncr can reach it locally.

4

Open the popup and go live

Click the Syncr icon, switch on the activities you want, and open a supported site. Your Discord profile updates straight away.

Full setup instructions live in the GitHub readme, including exact host install steps for Windows and Linux. If anything looks off, the repo issues page is the fastest way to get help.

Updates itself

Syncr checks for new versions and updates in place. Activity rules refresh from GitHub without a reinstall.

See what is live

The popup shows exactly what is being transmitted right now, so you are never guessing what your profile says.

Private by default

Turn any activity off and it never touches your profile. Nothing leaves your machine at any point.
